# frs-listener-data

Public output of the FRS Daily Language Listener. This repository holds a single aggregated file, `trends.json`, refreshed once daily by an automated pipeline. Every entry is aggregated across many public social posts and paraphrased by construction — no verbatim text, usernames, or per-post links are ever included. It is read-only input for downstream tooling and carries no warranty of completeness or accuracy.
